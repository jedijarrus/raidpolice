API-Credentials & Secrets

Hier liegen die Tokens für WCL und TMB. Werte werden niemals an den Client zurückgegeben — Feld leerlassen behält den gespeicherten Wert; Wert eintragen überschreibt ihn.

Warcraft Logs API v1 (Public Key)

Wo: warcraftlogs.com/profile → ganz unten „Web API → Public Key". Limit 600 req/h pro IP.

Warcraft Logs API v2 (OAuth Client)

Wo: warcraftlogs.com/api/clients/ → „Create Client" (Name beliebig, Redirect kann leer bleiben). Limit 9000 Punkte/h. Wird für Cast-Events benötigt.

ThatsMyBis Cookie

Wie extrahiert man den TMB-Cookie? ▾
  1. Browser öffnen, bei thatsmybis.com einloggen.
  2. DevTools öffnen (F12 oder Rechtsklick → „Untersuchen").
  3. Reiter „Application" (Chrome/Edge) bzw. „Speicher" (Firefox) → links unter „Cookies" → https://thatsmybis.com auswählen.
  4. Den Eintrag laravel_session finden — den Wert kopieren (lange Zeichenkette).
  5. Hier unten eintragen, im Format laravel_session=DER_WERT (das Prefix ist wichtig).

Der Cookie läuft nach einigen Wochen ab — wenn TMB-Daten 401/403 zurückgeben, hier neu setzen.
